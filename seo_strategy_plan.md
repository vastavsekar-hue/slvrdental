## 9. Technical SEO Checklist

Ensure your website performs well technically, as search engines down-rank slow or non-responsive websites.

1. **Website Speed Optimization (Core Web Vitals):**
   * **Images:** Compress all PNGs/JPGs in the `assets/images/` directory to modern WebP format. Set `loading="lazy"` on all offscreen images.
   * **Resource Deferral:** Add `defer` attributes to the script links (e.g., `<script src="scripts.js" defer></script>`).
   * **CSS/JS Minification:** Compress CSS (`styles.css`) and Javascript (`scripts.js`) to reduce load times.

2. **Mobile Responsiveness:**
   * Test your viewport tags. Your files correctly include `<meta name="viewport" content="width=device-width, initial-scale=1.0">`. Ensure CSS touch target sizes (buttons, navigation elements) are at least $48\text{px} \times 48\text{px}$.

3. **XML Sitemap Setup:**
   * Create an `sitemap.xml` file containing all static pages.
   * Submit the sitemap to **Google Search Console (GSC)**.

4. **Robots.txt Creation:**
   * Place a `robots.txt` file in the root directory to define access rules.
   ```text
   User-agent: *
   Allow: /
   Disallow: /admin/
   Disallow: /private/
   Sitemap: https://slvrdentalandcosmeticcare.in/sitemap.xml
   ```

5. **SSL Security (HTTPS):**
   * Ensure an active SSL certificate (HTTPS) is configured on the hosting server (Netlify, Hostinger, Vercel, or custom server).
